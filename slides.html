Etkili Prompt Yazma Rehberi

AI'ya ne kadar detaylı anlatırsanız,
o kadar iyi sonuç alırsınız.

✍️ 6 Altın Kural

  • 1️⃣ Net & Spesifik Olun — "Güzel bir site yap" yerine detay verin
  • 2️⃣ Yapıyı Belirtin — Bölümleri ve sırayı açıkça tarif edin
  • 3️⃣ Görsel Detay Verin — Renk (HEX), font, layout, bileşenler, görseller
  • 4️⃣ İçerik Sağlayın — Gerçek metin ve verileri aktarın
  • 5️⃣ Teknik Detaylar — Responsive, API, harici kütüphane kullanımı
  • 6️⃣ Referans Gösterin — Benzetmek istediğiniz siteleri belirtin

1️⃣ Net ve Spesifik Olun

❌ Kötü Prompt

Bana güzel bir web sitesi yap.

✅ İyi Prompt

Bir fotoğrafçı portfolyo sitesi oluştur. Tek sayfa olsun. Hero bölümünde büyük bir başlık ve arka plan görseli olsun. Altında masonry grid düzeninde galeri olsun.

💡 Ne kadar detay verirseniz, AI o kadar isabetli sonuç üretir. "Güzel" veya "modern" gibi sıfatlar yeterli değildir — somut özellikler belirtin.

2️⃣ Yapıyı Belirtin

Sayfanın bölümlerini sırasıyla tarif edin:

1) Hero: Tam ekran arka plan, slogan ve "Hakkımda" butonu 2) Hakkımda: Sol tarafta fotoğraf, sağ tarafta kısa biyografi 3) Projeler: 3 sütunlu grid, her kartta proje adı, açıklama ve link 4) İletişim: Basit form — isim, e-posta, mesaj 5) Footer: Sosyal medya ikonları ve telif hakkı

3️⃣ Görsel Detay Verin

🎨 Renkler (HEX)

Ana renk: #6366f1
Vurgu: #0ea5e9
Arka plan: #f8fafc

💡 coolors.co ile palet oluşturun

🔤 Fontlar

Başlıklar: Poppins (bold)
Gövde: Inter (regular)

💡 fonts.google.com

📐 Layout

3 sütunlu grid
Masonry düzen
Flexbox ile ortalama
Responsive breakpointler

3️⃣ Görsel Detay — Bileşenler & Görseller

🧩 UI Bileşenleri

  • Yuvarlak köşeli kartlar (border-radius: 12px)
  • Hover'da hafif yükselme efekti
  • Gradient arka planlı butonlar
  • Yapışkan (sticky) navigasyon
  • Hamburger menü (mobil)
  • Lightbox (fotoğraf büyütme)

🖼️ Görseller

  • Placeholder: "Renkli kutular kullan"
  • Unsplash: unsplash.com/photos/...
  • Picsum: picsum.photos/800/600
  • Emoji: 🎨 📸 🍕 ✈️
  • SVG ikon: Heroicons, Lucide
  • CSS ile: Gradientler, şekiller

✍️ Kurallar (devam)

4️⃣ İçerik Sağlayın

Gerçek metin, isimler, marka bilgileri verin. AI'ın uydurmasını beklemeyin.

"Firma adı: TechStart, slogan: Geleceği birlikte kodluyoruz"

5️⃣ Teknik Detaylar

  • Responsive / mobil uyumlu
  • Sadece HTML, CSS, JS
  • Harici kütüphane kullanma
  • API entegrasyonu

6️⃣ Referans Verin

"Apple.com gibi minimalist", "Notion.so gibi temiz tipografi", "Stripe.com gibi modern"

📱 Responsive Tasarım

Aynı site, her ekran boyutuna otomatik uyum sağlar

Mobil

Tek sütun düzen. İçerik alt alta dizilir.

Tablet

2 sütun düzen. Kartlar yan yana gelir.

Masaüstü

Çok sütunlu düzen. Tüm alanlar kullanılır.

💡 AI araçlarına "tamamen responsive olsun" veya "mobil uyumlu olsun" demeniz yeterli — media query'leri otomatik üretir.

📚 Bilmenizde Fayda Olan Bazı Kavramlar

📦 <div>

Kutu/kapsayıcı. İçerikleri gruplar.

🏷️ class / id

Etiketleme sistemi. CSS ile hedefleme.

📐 Flexbox

Öğeleri yatay/dikey hizalar.

🔲 Grid

2 boyutlu ızgara sistemi.

📱 Media Query

Ekran boyutuna göre farklı stil.

🔗 <a href>

Link oluşturmanın HTML yolu.

🔤 Font Boyutu & 📏 Boşluklar

Font Boyutu (CSS)

/* Başlıkları büyütmek */
h1 { font-size: 42px; }  /* → 48px */
h2 { font-size: 28px; }  /* → 32px */

/* Paragraf metni */
p { font-size: 16px; }   /* → 18px */

Boşluk Ayarları (CSS)

/* margin = dış boşluk */
.card { margin: 20px; }

/* padding = iç boşluk */
.card { padding: 24px; }

/* Sadece üstten boşluk */
.section { margin-top: 60px; }

🎨 Proje: Kişisel Portfolyo Sitesi

Kişisel portfolyo web sitesi oluştur. Tek sayfa (single page) olsun. Bölümler: 1) Hero/Tanıtım: İsim, unvan, kısa slogan, profil fotoğrafı placeholder 2) Hakkımda: 2-3 paragraf biyografi 3) Yetenekler: İkon veya progress bar ile göster (HTML, CSS, JS, Photoshop vb.) 4) Projeler: 3-4 adet proje kartı (grid layout). Her kartta: - Proje görseli placeholder, Proje adı, Kısa açıklama, "Demo" ve "GitHub" linkleri 5) İletişim: Basit form (İsim, E-posta, Mesaj) + sosyal medya linkleri 6) Footer: Telif hakkı, sosyal linkler Tasarım: Modern, minimalist. Ana renk: #6366f1 (indigo). Karanlık tema. Font: Google Fonts - "Inter". Smooth scroll, mobil uyumlu.

📝 Proje: Kişisel Blog

Kişisel blog web sitesi oluştur. İki sayfa olsun: Ana sayfa ve Yazı sayfası. Ana Sayfa: - Üstte navigasyon (Blog adı, Hakkımda, Kategoriler) - Hero: Blog adı, kısa açıklama - Blog yazıları listesi: Kart düzeninde, her kartta tarih, kategori, başlık, özet, "Devamını Oku" - Sidebar: Hakkımda kutusu, Kategoriler listesi, Son yazılar Yazı Sayfası: - Başlık, yazar, tarih, okuma süresi - Makale içeriği (güzel tipografi ile) - Fotoğraf alanı, detaylı biyografi, sosyal medya linkleri Tasarım: Temiz, okunabilir, beyaz arka plan, tipografi odaklı. Ana renk: #059669 (yeşil). Mobil uyumlu. Örnek blog yazıları (3 adet): teknoloji, seyahat ve yemek konularında.

🍽️ Proje: Restoran / Kafe Sitesi

Bir kafe/restoran için tanıtım web sitesi oluştur. Tek sayfa olsun. İşletme Bilgileri: - İsim: "Kahve Durağı", Slogan: "Her yudumda bir hikaye" - Adres: İstanbul, Kadıköy | Çalışma saatleri: 08:00 - 22:00 Bölümler: 1) Hero: Tam ekran arka plan görseli, kafe adı ve sloganı 2) Hakkımızda: Kısa hikaye, kuruluş yılı 3) Menü: Kategorilere ayrılmış (Kahveler, Tatlılar, Atıştırmalıklar) 4) Galeri: 6 adet fotoğraf placeholder'ı grid düzeninde 5) Konum ve İletişim: Google Maps placeholder, adres, telefon 6) Footer: Sosyal medya ikonları Tasarım: Sıcak renkler (kahverengi, krem tonları), zarif font. Mobil uyumlu.

📄 Proje: Online CV / Özgeçmiş

Online CV (özgeçmiş) web sayfası oluştur. Tek sayfa, A4 kağıt görünümünde. Sol Sütun (koyu arka plan): - Profil fotoğrafı (yuvarlak), isim ve unvan - İletişim bilgileri (e-posta, telefon, konum) - Beceriler (progress bar: Photoshop %90, Illustrator %85, Figma %80) - Yabancı Dil (İngilizce: İleri, Almanca: Orta) Sağ Sütun (açık arka plan): - Profil/Özet paragrafı - İş Deneyimi (3 adet), Eğitim (2 adet), Sertifikalar Tasarım: Profesyonel, temiz. Yazdırılabilir olsun (print CSS). Renk: koyu lacivert + beyaz. Mobilde tek sütun. Kişi: "Mehmet Kaya, Dijital Pazarlama Uzmanı"

📱 Proje: Ürün Tanıtım Sayfası

Mobil uygulama için landing page oluştur. Uygulama: "FitTakip" — Egzersiz ve beslenme takip uygulaması Bölümler: 1) Hero: Başlık, açıklama, "Hemen İndir" butonu + telefon mockup (CSS ile) 2) Özellikler: 3 sütun — "Egzersiz Takibi", "Kalori Sayacı", "İlerleme Grafikleri" 3) Nasıl Çalışır: 3 adım (1-Kaydol, 2-Hedef Belirle, 3-Takip Et) 4) Kullanıcı Yorumları: 3 testimonial kartı 5) Fiyatlandırma: 2 plan (Ücretsiz, Premium — 49.99 TL/ay) 6) SSS: 4-5 accordion soru-cevap 7) CTA: "Bugün başla", e-posta giriş alanı Tasarım: Gradient (mor-mavi), modern, enerjik, animasyonlu, responsive.

GitHub Pages ile ücretsiz yayınlama

🐙 GitHub Pages — 4 Adım

1

GitHub Hesabı Oluşturun

github.com adresine gidin ve ücretsiz bir hesap açın.

2

Repository Oluşturun

Sağ üstteki + → "New repository". İsim: kullaniciadi.github.io

3

Dosyalarınızı Yükleyin

"uploading an existing file" linkine tıklayın → dosyaları sürükleyip bırakın → "Commit changes"

4

Siteniz Yayında! 🎉

kullaniciadi.github.io formatındaki repolar otomatik etkindir. Birkaç dakika bekleyin!

🐙 GitHub Pages — Önemli Detaylar

👤 Kullanıcı Adınızı Bulma

  • Sağ üst profil fotoğrafına tıklayın
  • Profil URL'niz: github.com/kullaniciadi
  • Repo oluştururken Owner alanında görünür

📁 Dosya Yükleme

  • İlk yükleme: "uploading an existing file" linki
  • Sonraki yüklemeler: "Add file" → "Upload files"

Frontend vs Backend

Web sitelerinin iki temel katmanı

🔄 Frontend vs Backend

🎨 Frontend (Ön Yüz)

HTML + CSS + JavaScript — Tarayıcıda çalışır

  • Kullanıcının gördüğü her şey
  • Tasarım, renkler, animasyonlar
  • Buton tıklamaları, menüler, mobil uyumluluk

⚙️ Backend (Arka Yüz)

Node.js, Python, Veritabanı — Sunucuda çalışır

  • Kullanıcının görmediği kısım
  • Veritabanı işlemleri, güvenlik
  • E-posta, ödeme, veri analizi

🍽️ Restoran benzetmesi: Frontend = yemek salonu | Backend = mutfak. AI ile yaptığımız siteler sadece "salon" kısmıdır — çoğu proje için yeterli!

📄 Statik Site vs ⚙️ Dinamik Site

📄 Statik Site (Sadece Frontend)

HTML, CSS ve JS dosyaları olduğu gibi gönderilir.

Örnekler: Portfolyo, Blog, CV, Düğün Davetiyesi, Ürün Tanıtımı

  • ✅ GitHub Pages ile ücretsiz
  • ✅ Süper hızlı (CDN)
  • ✅ Güvenlik riski minimal
  • ✅ AI araçları ile kolayca yapılır

⚙️ Dinamik Site (Frontend + Backend)

Sunucuda kod çalışır, veritabanından veri çekilir.

Örnekler: E-ticaret, Sosyal Medya, Bankacılık

  • ⚠️ Sunucu ve veritabanı gerekir
  • ⚠️ Daha karmaşık ve maliyetli
  • ⚠️ Güvenlik bilgisi önemli
  • ⚠️ Bakım ve güncelleme gerekir

✅ Yapabilecekleriniz & ❌ Yapamayacaklarınız

✅ Statik Site ile

  • Kişisel web sitesi / Portfolyo
  • Blog yazıları
  • İletişim formu (Formspree ile)
  • Yorum sistemi (Giscus ile)
  • Fotoğraf galerisi & Etkinlik sayfası
  • Veri görselleştirme (CSV/JSON)

❌ Statik Site ile

  • Kullanıcı kayıt/giriş sistemi
  • Veritabanına veri kaydetme
  • E-posta gönderme
  • Ödeme alma
  • Gerçek zamanlı sohbet

🔧 Backend Olmadan Çözümler

💾 localStorage

Tarayıcıda küçük veri saklama (tema, checklist durumu)

📬 Formspree

İletişim formlarından e-posta alma. Aylık 50 ücretsiz.

💬 Giscus

GitHub Discussions tabanlı ücretsiz yorum sistemi.

🔐 Auth0 / Firebase Auth

Google, GitHub ile giriş. 7.000 kullanıcı ücretsiz.

📊 Google Sheets API

Google Sheets'i basit veritabanı olarak kullanın.

🍪 Cookies

Küçük metin verileri. Dil tercihi, bildirimler.

🗄️ Ücretsiz Veritabanı Servisleri

🔥 Firebase Firestore

Google'ın NoSQL veritabanı. Gerçek zamanlı senkronizasyon.

  • 1 GB depolama
  • 50K okuma/gün
  • 20K yazma/gün

⚡ Supabase

Açık kaynak Firebase alternatifi. PostgreSQL.

  • 500 MB veritabanı
  • 1 GB dosya depolama
  • 50K aktif kullanıcı/ay

🍃 MongoDB Atlas

En popüler NoSQL. JSON benzeri belgeler.

  • 512 MB depolama
  • Esnek şema
  • MongoDB Compass

🖥️ Tam Backend İstiyorsanız

Bu platformlar ücretsiz başlangıç katmanları sunar:

🚂 Railway

Node.js, Python, Go. GitHub'dan otomatik deploy. Aylık $5 kredi.

🎨 Render

Statik siteler, web servisleri, veritabanları. Ücretsiz katman.

🪁 Fly.io

Docker container dağıtımı. 3 paylaşımlı VM ücretsiz.

⚡ Replit

Tarayıcıdan tam backend. AI destekli kodlama dahil.
